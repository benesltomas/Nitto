SwitchConf - Error numbers

0 OK Function OK
50001 INVALID_AXIS_REFERENCE Reference information for axis invalid
50002 INVALID_DEVICE_NAME Device name for switch invalid
50003 INVALID_CHANNEL_NAME Channel name for switch invalid
50004 ERROR_GET_OFFSET_FOR_SWITCH Error getting offset for switch
50005 ERROR_WRITE_PARAMETER_LIST Error writing parameter list to axis
65535 FUNCTION_BUSY Function working

Look for description of other error numbers in the help file of the libraries Acp10Man, Acp10_MC or powerlnk.
