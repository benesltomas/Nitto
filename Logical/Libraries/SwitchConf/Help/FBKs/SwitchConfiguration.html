SwitchConf - SwitchConfiguration()

This function block configures end-switches and reference switch for APCm.

HINT:

Modules were switches are connected must be in the POWERLINK network.

If you don't set a device name for the switch it is skipped.

Parameters:

I/O

Parameter

Data Type

Description
IN Axis UDINT axis reference
IN Execute BOOL start initialization at rising edge
IN posEndSwitchDeviceName UDINT device name of module where positive end switch is connected
IN posEndSwitchChannelName UDINT channel name of i/o where positive end switch is connected
IN negEndSwitchDeviceName UDINT device name of module where negative end switch is connected
IN nedEndSwitchChannelName UDINT channel name of i/o where negative end switch is connected
IN refSwitchDeviceName UDINT device name of module where reference switch is connected
IN refSwitchChannelName UDINT channel name of i/o where reference switch is connected
OUT Done BOOL initialization complete
OUT Busy BOOL initialization active
OUT Error BOOL error occured in FB
OUT ErrorID ErrorNumbers_typ error number

Call Syntax (Structured Text):

SwitchConfiguration_Lift1.Axis := ADR(g211A1);
SwitchConfiguration_Lift1.Execute := TRUE;
SwitchConfiguration_Lift1.posEndSwitchDeviceName := ADR ('IF6.ST15.IF1.ST7');
SwitchConfiguration_Lift1.posEndSwitchChannelName := ADR ('DigitalInput02');
SwitchConfiguration_Lift1.negEndSwitchDeviceName := ADR ('IF6.ST15.IF1.ST7');
SwitchConfiguration_Lift1.negEndSwitchChannelName := ADR ('DigitalInput01');
SwitchConfiguration_Lift1.refSwitchDeviceName := 0;
SwitchConfiguration_Lift1.refSwitchChannelName := 0;
SwitchConfiguration_Lift1();
